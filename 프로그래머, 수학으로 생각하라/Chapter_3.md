# 나머지: 주기성과 그룹 나누기

1. **예제**
    * **N일 뒤의 요일**: (일 수) % 7 => [일, 월, 화, 수, 목, 금, 토]
    * **10^N일 뒤의 요일**: N % 6 => [월, 수, 화, 토, 목, 금]
    * **N의 거듭제곱 1의 자리**: N의 1의 자리를 거듭제곱하여
    * **오셀로 게임**: 마술사의 트릭 -> 검은색 돌이 홀수가 되면 뒤집음, 짝수면 아님

2. **패리티 확인**
   * (오셀로 게임) 송신자: 조수, 수신자: 마술사, 노이즈: 관객
   * 하나의 돌을 놓음으로 노이즈가 있는지 없는지 판단
